# Personal Portfolio Website

This portfolio website was created to present my academic journey, technical interests, and growing project experience as an Electrical and Electronics Engineering student. It highlights my work in programming, embedded systems, electronics, and my curiosity toward VLSI, hardware design, and intelligent computing.

The website includes sections for my introduction, skills, projects, achievements, and contact information, designed to serve as both a personal showcase and a record of my learning journey. It was built with a clean and organised structure so it can be updated over time as I continue exploring new technologies and building more projects.
 
## Purpose
- Showcase my technical profile and projects
- Present my interests in electronics, embedded systems, and computing
- Create a central space for future academic and professional work

## Built With
- HTML
- CSS
- JavaScript

## Status
This portfolio is a continuing project and will evolve as I learn, build, and refine my work.
